
# 스프링 전용 IDE 설치

	1. STS (이클립스 기반)
	2. Visual Studio Code
	3. IntelliJ
	
# 스프링 부트 프로젝트 만들기

	1. File - New - Spring Starter Project
	
	2. 프로젝트 정보 설정
	
		(1) Name : 프로젝트 이름
		(2) Type : 프로젝트 관리 도구 선택 (Maven, Gradle)
		(3) Packaging : 프로젝트 빌드 결과물의 확장자
		(4) Java Version : 자바 버전
		(5) Language : 사용하고 싶은 프로그래밍 언어(Java, Kotlin, Groovy)
		(6) Group : Maven 그룹 이름(주로 회사명 사용)
		(7) Artifact : Maven 프로젝트 이름
		(8) Version : 빌드 결과물의 버전 이름
		(9) Desciption : 프로젝트에 대한 설명
		(10) Package : 우리가 사용할 기본 패키지 이름
		
	3. 의존성 선택
	
		- 의존성 : 이 프로젝트를 만들기 위해 사용될 다른 프로젝트(라이브러리)들
		- Spring Web : 웹 프로젝트에 MVC 디자인 패턴을 쉽게 적용할 수 있는 프레임워크
		- Thymeleaf : JSP의 쓸데없는 기능을 모두 제거한 뷰 라이브러리(대부분의 기능이 제거되고 뷰를 생성하는 데 필요한 EL, JSTL과 비슷한 기능만 남아있다.)
		
# 프로젝트 실행해보기

	- 생성한 프로젝트 이름으로 된 클래스에서 main 메서드를 실행한다.(e.g. 프로젝트 이름이 SpringMvc일 때 SpringMvcApplication)
	- 포트 번호가 겹치는 경우 src/main/resources/application.properties 파일에서 포트번호를 수정할 수 있다.
